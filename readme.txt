== Description ==

The Payment Gateway Remise for WooCommerce plugin adds the functionality to take Remise payments on your store of WooCommerce.

= About Remise =

Remise is a comprehensive payment platform that offers various online payment methods, such as credit card payment and convenience store payment in Japan.

[Remise](http://www.remise.jp/)

In order to use Remise, you have to purchase the authentication key at the following site.

[WordPress Market](https://www.wpmarket.jp/)

== Installation ==

1. Upload the plugin file in the plugin install page or copy the `wc-remise-gateway` directory into your `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! :)
